# MindTrace

MindTrace is a lightweight mental health pattern tracker focused on
self-awareness and behavioral trends, not diagnosis or therapy.

## What it does
- Logs daily mental health indicators
- Stores data securely in CSV format
- Analyzes long-term patterns
- Highlights potential risk trends

## Metrics tracked
- Mood (1–10)
- Sleep hours
- Stress level (1–10)
- Screen time
- Optional notes

## GUI Version
MindTrace includes a simple Tkinter-based desktop GUI.
